🧾 Caderno de Fiados

Este projeto é uma versão digital e simples do famoso “caderninho de fiados”, feita em Python e pensada especialmente para pequenos comércios. A ideia é transformar aquele controle feito no papel em algo mais organizado e prático, direto no computador.

🎯 Qual é a ideia?

A proposta aqui é ajudar donos de mercearias, quitandas e outros pequenos negócios a terem um jeito rápido e fácil de acompanhar as dívidas dos clientes pelo terminal. Com esse programinha, você consegue:

Ver quem está devendo.

Registrar novas compras feitas “no fiado”.

Dar baixa quando alguém paga.

Consultar saldos sem complicação.

✨ O que o sistema faz

Cadastro de Clientes:

Adicionar novos clientes.

Remover clientes que não precisam mais estar no sistema.

Movimentações de Dívidas:

Registrar novas compras, aumentando o valor devido.

Registrar pagamentos, diminuindo o saldo.

Consultas e Relatórios:

Ver o extrato completo de um cliente específico.

Listar todos os clientes com seus respectivos débitos.

Mostrar o total geral que a loja tem a receber.

🛠️ O que foi usado para construir

Linguagem: Python 3

Armazenamento: Arquivo JSON local (dados.json) para guardar os dados.

Interface: Tudo funciona no terminal, com um menu simples e direto.

🚀 Como rodar o projeto

Verifique se o Python 3 está instalado no seu computador.

Baixe ou clone o repositório.

Abra o terminal na pasta onde o projeto está salvo.

Rode o arquivo principal com o comando:

python caderno_de_fiados.py

No primeiro uso, o arquivo dados.json será criado automaticamente.
